# 🏦 Bankify - Frontend (React)

A modern frontend for a full-stack banking application built with React.  
This application allows users to securely manage their bank accounts, perform transactions, and view transaction history.

## 🛠 Tech Stack
- React (Frontend)
- Tailwind CSS (UI Styling)
- REST APIs (Integration with Spring Boot backend)
- Vercel (Deployment)

---

## ✨ Features
- 🔐 User Authentication (Login / Signup)
- 💰 Deposit & Withdraw Money
- 🔄 Transfer Money Between Users
- 📜 Transaction History
- 📊 Dashboard with Live Balance
- 🎨 Clean & Responsive UI
